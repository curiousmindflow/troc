# Troc
Troc is a Data Distribution Service implementation in Rust, using [sans-io](https://sans-io.readthedocs.io/) for the protocol domain, designed for reliability, testability and high performance.

## Why Troc ?
Other DDS implementation in Rust exists and are production ready (RustDDS, DustDDS) but none of them, even those written in another language, uses the [sans-io](https://sans-io.readthedocs.io/) architectural pattern.
Using the sans-io pattern enable a substantial number of benefits while not bringing so much downsides.

## Project status
Troc is in early stage and is not production ready. It's still experimental on some aspects.
